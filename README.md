Desenvolver um sistema de cadastro de filmes em laravel com uma dashboard administrativa, deve ter tela de login com usuário e senha e para os campos ter imagem, sinopse e afins.

Tabela filme:

-   Título
-   Sinopse
-   Capa
-   Direção
-   Data de lançamento
-   Nota
-   Principal categoria

Tabela Categorias:

-   Ação
-   Aventura
-   Animação
-   Drama
-   Romance
-   Comédia
-   Fantasia
-   Terror
-   Ficção científica
-   Musical
-   Faroeste
-   Documentário
